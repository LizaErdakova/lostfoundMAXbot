# Demo photos

Изображения в этой папке используются скриптом `server/src/seed.js`, чтобы заполнять демонстрационные объявления с фотографиями.  

- замените PNG/JPEG своими фото и закоммитьте их в репозиторий, чтобы у всех участников были те же примеры;
- пути к файлам должны начинаться с `/sample/...` (например, `/sample/pet-mona.png`);
- после обновления фото перезапустите `npm run seed` в директории `server`, чтобы пересоздать базу с новыми картинками.
